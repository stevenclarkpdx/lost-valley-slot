# Lost Valley Slot Prototype

A browser-based 2D slot design sandbox for exploring slot math, feature volatility, and presentation feel.

The project is not a real-money gambling product. It is a simulation/design tool.

Core goals:
- 5x5 expedition-themed main board.
- Feature trigger from Footprint scatters.
- Three Hold & Spin feature variants: Fossil Valley, Predator Valley, Nesting Grounds.
- Tweakable math parameters.
- Manual play mode and bulk simulation mode.
- Visual presentation should feel like a gritty commercial slot cabinet, not a clean web app.
